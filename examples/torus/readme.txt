Description: The torus example demonstrates the usage of the libfit library
	on a torus segment.

Directory: examples/torus
Subdirectories:
	output/: contains new fine meshes

Instructions:
	1. Generate coarse mesh: compile generator.f90 and execute.
	2. Generate new fine mesh using libfit: execute main
	3. Visualize new fine mesh: configure and execute vis.py
